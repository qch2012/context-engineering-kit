# Additional Resources

Welcome to the Context Engineering Kit resources section. This collection provides supplementary materials, external references, and community resources to help you get the most out of CEK.

## What's in This Section

### [Related Projects](./related-projects.md)

Discover the ecosystem of projects that inspired and complement Context Engineering Kit:

- Official Claude Code plugins and extensions
- Community-driven task management and orchestration systems
- Example implementations and skill libraries
- Spec-Driven Development frameworks

Learn where CEK draws its inspiration and how it integrates the best ideas from multiple sources.

### [Recommended MCP Servers](./recommended-mcp.md)

Explore Model Context Protocol servers that extend Claude Code's capabilities:

- **Context7** - Load documentation for specific technologies
- **Serena** - Semantic code retrieval and editing
- **Perplexity MCP** - Enhanced search and research capabilities

These servers complement CEK plugins by providing external knowledge access and specialized tooling.

### [Glossary](./glossary.md)

A comprehensive reference of terms and concepts used throughout Context Engineering Kit:

- Core terminology (plugins, commands, skills, agents)
- Context engineering concepts
- Development methodology terms
- Technical specifications

Use this as a quick reference when encountering unfamiliar terms in documentation.

### [FAQ](./faq.md)

Answers to frequently asked questions:

- Installation and setup
- Plugin selection and usage
- Troubleshooting common issues
- Best practices and optimization
- Integration with workflows

Start here if you have questions about getting started or resolving common issues.

## How to Use These Resources

### For Beginners

If you're new to Context Engineering Kit:

1. Start with the [FAQ](./faq.md) to understand common questions and answers
2. Review the [Glossary](./glossary.md) to familiarize yourself with terminology
3. Explore [Related Projects](./related-projects.md) to understand the broader ecosystem

### For Experienced Users

If you're already familiar with CEK:

- Check [Recommended MCP Servers](./recommended-mcp.md) to extend your capabilities
- Browse [Related Projects](./related-projects.md) for inspiration and complementary tools
- Reference the [Glossary](./glossary.md) when documenting your own workflows

### For Contributors

If you're contributing to CEK:

- Study [Related Projects](./related-projects.md) to understand prior art and influences
- Use the [Glossary](./glossary.md) to maintain consistent terminology
- Review the [FAQ](./faq.md) to identify common pain points
