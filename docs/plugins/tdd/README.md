# Test-Driven Development Plugin

> **Status:** Documentation in progress

## Overview

Commands and skills for test-driven development with anti-pattern detection. Introduces TDD methodology, best practices, and skills for testing using subagents.

## Installation

```bash
/plugin install tdd@NeoLabHQ/context-engineering-kit
```
