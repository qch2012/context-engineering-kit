# Docs Plugin

> **Status:** Documentation in progress

## Overview

Commands for project analysis and documentation management. Introduces commands for analyzing projects, writing and refining documentation.

## Installation

```bash
/plugin install docs@NeoLabHQ/context-engineering-kit
```

## Quick Info

**Plugin Type:** Commands
**Key Features:**
- Project analysis
- Documentation writing and refinement
- Implementation documentation updates
