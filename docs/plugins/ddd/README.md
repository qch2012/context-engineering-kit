# Domain-Driven Development Plugin

> **Status:** Documentation in progress

## Overview

Commands for setting up domain-driven development best practices focused on code quality. Includes Clean Architecture, SOLID principles, and other design patterns.

## Installation

```bash
/plugin install ddd@NeoLabHQ/context-engineering-kit
```

## Quick Info

**Plugin Type:** Commands and Skills
**Key Features:**
- Clean Architecture principles
- SOLID design patterns
- Code formatting guidelines
- Software architecture best practices
