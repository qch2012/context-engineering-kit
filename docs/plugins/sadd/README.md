# Subagent-Driven Development Plugin

> **Status:** Documentation in progress

## Overview

Skills for subagent-driven development with quality gates between tasks. Dispatches fresh subagent for each task with code review between tasks, enabling fast iteration with quality gates.

## Installation

```bash
/plugin install sadd@NeoLabHQ/context-engineering-kit
```

## Quick Info

**Plugin Type:** Skills
**Key Features:**
- Subagent dispatch for independent tasks
- Code review between tasks
- Fast iteration with quality gates
