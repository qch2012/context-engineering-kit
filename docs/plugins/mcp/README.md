# MCP Plugin

> **Status:** Documentation in progress

## Overview

Commands for integrating Model Context Protocol servers with your project. Introduces commands for setting up well-known MCP server integration if needed and updating CLAUDE.md file with requirements to use this MCP server for the current project.

## Installation

```bash
/plugin install mcp@NeoLabHQ/context-engineering-kit
```

## Quick Info

**Plugin Type:** Commands
**Key Features:**
- Context7 MCP server setup
- Serena MCP server setup
- Custom MCP server creation guide
- CLAUDE.md integration for MCP requirements

## Available Commands

- `/mcp:setup-context7-mcp` - Guide for setup Context7 MCP server to load documentation for specific technologies
- `/mcp:setup-serena-mcp` - Guide for setup Serena MCP server for semantic code retrieval and editing capabilities
- `/mcp:build-mcp` - Guide for creating high-quality MCP servers that enable LLMs to interact with external services
